# Layout de Login do Instagram

### Descrição:

- Fiz um clone da página de login do Instagram, utilizando o flexbox do CSS. Trabalho feito como desafio de código da DIO, com layout responsivo.

### Tecnologias utilizadas:

- HTML
- CSS

### Deploy do site:
